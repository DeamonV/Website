Über das Projekt

Deamon-V ist ein GTA-V Roleplay Server für jedermann! 

Schau dich ersteinmal hier auf der Website um. Später kannst du ja immernoch den Discord Server betreten, aber das ist ganz dir überlassen.

Du hast noch Fragen? Dann schau mal auf unserer FaQ Seite vorbei. Wenn du deine Frage nicht finden kannst, dann frage bitte auf unserem Discord Server nach. 

Weiterhin wünscht dir dass gesamte Deamon-V Team viel spaß, sowohl hier, auf dem Discord Server, FiveM Server oder Minecraft Server.
